Instructional Design

A graduate course in designing learning experiences for adults. Taught by Karl Stolley, Fall Semester 2012 at Illinois Institute of Technology.

Course Project

The Project For lack of a better name

Description

This is the one and only, class-wide collaborative project for COM535: Instructional Design. For The Project, we (including the instructor) will be designing a learning experience for students in our graduate program. The project will enable self-guided learning or instructor-lead teaching of some fundamental digital technologies that all students in the program should know.

The project itself will be in the form of a web-available resource that can function both as a teach-yourself resource, or as the resource for a class. One of the deliverables for the project is the development of an 8- and 16-week syllabus based on the resource.

Deliverables

The Project will include the following deliverables. Additional deliverables (such as learner testing) will be added based on class discussions & progress. Watch this space.

  • The main stuff:
    1. A complete, open instructional resource available at (URL TBD) featuring a cohesive design & user experience, but covering each student’s subject area.
    2. Two syllabi, one lasting 8 weeks and one lasting 16 weeks, for using the resource in a more formal (workshop, class) setting.
  • At each assessment point, each individual student will deliver:
    1. A 500-word report memo detailing contributions and problems to the subject area of the student’s responsibility, and to others’ subject areas/the project as a whole; both contributions and problems should refer to specific commits in our GitHub repository.
    2. An oral presentation to the other members of the class, including walk-throughs of sketches, actual design work, and any other materials produced for the project.
  • At the final assessment point:
    1. The completed resource
    2. The completed 8- and 16-week syllabi
    3. A presentation to an audience outside of our class (?)
    4. Announcement of the resource's availability on social media channels, etc.

Assessment Points

There is only this one project in the class. But rather than waiting until the final night of class to submit the project & receive a grade, grading will be based on five different assessment points: September 11, September 25, October 9, October 30, and December 4. The majority point-values of these assessment points are earned by mid-semester, with a mere 10 points remaining to be decided the final night of class.
