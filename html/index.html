To Get The Most Out of This Tutorial

You should...

  • Be familiar with text editors and browsers (if not go here
  • Have reviewed the Markdown tutorial with reasonable understanding are ready to move on to the next level

Learning Objectives

  • Learn how to create a basic web page.
  • Become familiar with HTML5 tags, attributes, and elements.
  • Understand what each of the following “tags" does and how to use them.
    • <!DOCTYPE html>
    • <html>
    • <head>
    • <title>
    • <meta>
    • <link>
    • <body>
    • <header>
    • <article>
    • <section>
    • <footer>
    • <a>
    • <img>
    • <table>
    • <p>
    • <nav>
    • <ol>
    • <ul>
    • <li>
    • <row>
  • Become familiar with each of the following “tags” for future reference and projects.
    • <aside>
    • <audio>
    • <details>
    • <output>
