Structure

Body

The body of the webpage houses all of the content that is intended to be displayed to the public. The browser renders everything enclosed within the <body> and </body> tags so that it is viewable. All other tags that we learn about go inside the body of the document.

Tags Used Within the Body

Div tag

The <div> is a generic container that is used when none of the other tags appropriately define the information contained. They are used more often than not with CSS.

Section

The <section> tag is used to group like content together. Sections could be nested or stand alone. Usually, the sections contain headers and footers.

Article

The <article> tag is used for grouped information within sections. These are usually not nested unless they are nested within sections of other sections [aside: to see more about nesting, go to Nesting].

Paragraph

The <p> tag is probably the most common tag you will see in a document, simply because it defines the portion of text that should be displayed as a paragraph. It is possible for a very simple page to be composed simply of <p> </p> elements, however, it is very limited and it doesn't tell you much about the information enclosed. Also, when we start to style the page using CSS, other tags, such as section and article help better define parts of the webpage.

Navigation

The <nav> </nav> tag groups navigation links together. These links are usually presented as an unordered list (see below). [to see more about links go to Links and Images].

Footer

The footer comes at the end of a section or the page with extra information.
