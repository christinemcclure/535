Tables

Tables are a series of rows and columns that create cells in which data lives.

Tables are built using the <table> and </table> tags. Anything within these tags are part of the table. Tables are built row by row using the <row> and </row> tags. The <table><table> tags indicate the top row heading, which, by default, is bold (can be changed with CSS). The <table> <table> tags contain the table cell data. Our example has a table. Each individual data cell are housed within the <td> </td> tags.s

thead, tbody, and tfoot

Although a table could be built with just the 3 tags, 2 really, if you want to keep it very simple, HTML5 also provides three tags that break the table into natural sections for styling purposes [aside: use CSS]. The <thead>and </thead> tags enclose the data in the first section to define the heading row of the table. Similarly, the <tbody>and </tbody> and the <tfoot>and </tfoot> tags enlose and define the body and footer data respectively.
